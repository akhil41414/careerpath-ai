ℹ️ About this project
Stack: Vanilla HTML/CSS/JS + Node.js serverless backend (Vercel)
Database: Supabase PostgreSQL — users, profiles, all analysis results
AI Model: Llama 3.3 70B via Groq API — server-side, key never exposed
Auth: Custom JWT tokens + bcrypt password hashing
Email alerts: Resend API — admin notified on every signup and login
Deployment: Vercel free tier with GitHub auto-deploy
