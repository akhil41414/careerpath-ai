About this project
Stack: HTML/CSS/JS frontend + Node.js Vercel serverless backend
Database: Supabase (PostgreSQL) — stores users + all analysis results
AI: Google Gemini 2.0 Flash via server-side proxy — API key never exposed to browser
Auth: JWT tokens — bcrypt password hashing, tokens expire in 7 days
Admin alerts: Resend email API — you get notified on every signup and login
Deployment: Vercel (free tier) — auto-deploy from GitHub on every push
